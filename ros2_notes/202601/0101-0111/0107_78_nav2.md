- ros2 run nav2_map_server map_server_cli -f : 保存rviz中的地图，通过订阅map话题获得最新的地图。

- navigation 2是开源的机器人导航框架，让机器人能自主移动
    - 路径规划、避障和自主脱困，是navigation 2所具备的基本能力
    - 使用行为树（Behavior Tree，BT）

- navigation编程指南
    - 使用话题初始化起点位姿
    - 使用TF获取机器人实时位置
    - 调用nav2接口进行单点导航
        - 动作通信：主要用于控制场景，优点是反馈机制
        - 使用命令行工具发送动作请求并接收反馈和结果，请求机器人移动到地图的指定目标点，使用python代码演示
            - 在nav2_simple_commander库已将动作客户端封装到了BasicNavigator节点中
    - 调佣nav2接口完成路点导航

- 使用python通过调用nav2_simple_commander库方便地实现导航，若项目需要C++，则使用动作客户端实现导航接口调用。
