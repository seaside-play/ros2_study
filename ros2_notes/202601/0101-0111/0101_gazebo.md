# 0 机器人建模与仿真————创建自己的机器人

在算法和程序部署到真实的机器人之前，往往会采用仿真的机器人进行验证。

在仿真之前，需要先对仿真的机器人进行建模

# 1 建模和仿真概述

## 1.1 机器人的结构介绍

- 测量距离的激光雷达、超声波和深度相机传感器
- 测量轮子速度的轮式编码器
- 测量加速度信息的惯性传感器
- 图像、电量和红外等创建传感器

举例：控制系统通过超声波传感器获取前方有障碍物的信息，此时就可以控制轮子停下并转向。

因为控制系统通常是计算机或微控制器来实现的， 所以，对于仿真机器人来说，只需要对机器人的传感器、执行器以及环境进行仿真即可。

仿真：就是利用软件来模拟硬件，在机器人中指的是各种执行器、传感器以及环境。

环境仿真：是机器人的作业场景和作业对象的仿真，常见的有工厂、居住环境，甚至水底。

# 1.2 仿真平台

- Gazebo: ROS 开源社区出生，支持多种传感器，执行器，以及模拟机器人和各种常用环境的运动和交互
- Webots： 自动驾驶仿真机器人
- CoppeliaSim：
- Unity3D：
- Matlab/Simulink:
